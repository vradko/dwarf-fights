# Dwarf Fights

### Random fighter picker disguised as an octagon riot

A randomizer at heart -- enter any group of names and let the octagon decide who wins. Every name gets a generated dwarf identity, a ridiculous weapon, a color palette, combat stats, and one job: survive the pileup. Use it to pick a restaurant, settle an argument, or just watch dwarves beat each other up.

![Hero Compilation](hero-compilation.jpg)

---

## What is this?

A zero-install browser game. Enter 2-50 names, hit "Fight", and watch procedurally generated dwarf fighters beat each other up in an animated octagon. All visuals are generated locally -- no assets to download, no server needed.

## Features

- **2 to 50 fighters** -- free-for-all brawls in a single octagon
- **Procedural dwarf generation** -- each name gets a unique sprite, color palette, weapon, and fighting style
- **PixiJS + Matter.js** -- WebGL rendering with real 2D physics collisions
- **Balanced combat system** -- dodge, glancing blows, critical hits; no more one-shot kills
- **Berserker mode** -- last 2 fighters alive enter rage mode to break stalemates
- **200 random trash facts** -- each fighter comes with a useless but entertaining bio fact
- **Beer bottle cap celebration** -- winner gets a shower of realistic 3D SVG bottle caps with a pop sound
- **Adjustable speed (0.2x - 5x)** -- slow-mo replays or blitz through the chaos

## Play

1. Open `index.html` in any modern browser, or
2. **[Play live on GitHub Pages](https://vradko.github.io/dwarf-fights/)**

No build tools, no dependencies -- just open and go.

## Tech Stack

- **PixiJS v7** -- WebGL/Canvas 2D rendering
- **Matter.js** -- 2D physics engine
- **Vanilla JS** -- no frameworks, no bundlers
- **CSS3** -- UI and animations
